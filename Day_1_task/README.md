# Day_1_Task

Day 1 self-practice — Python fundamentals covering variables, loops, functions, lists, dictionaries, exception handling, and basic problem-solving. Based on the AIML Crash Course Day 2 Practice Tasks.

## Files

| File                  | What it does                                                                          | How to run                   |
| --------------------- | ------------------------------------------------------------------------------------- | ---------------------------- |
| `intro.py`            | Prints a structured self-introduction using variables, dictionaries, and f-strings    | `python intro.py`            |
| `skills_counter.py`   | Displays a numbered list of skills and prints the total count                         | `python skills_counter.py`   |
| `even_odd.py`         | Accepts user input and determines whether the number is even, odd, or zero            | `python even_odd.py`         |
| `tip_calculator.py`   | Calculates tip amount and total bill using reusable functions                         | `python tip_calculator.py`   |
| `word_frequency.py`   | Counts word occurrences in a sentence and displays frequency statistics               | `python word_frequency.py`   |
| `calculator.py`       | Interactive calculator supporting addition, subtraction, multiplication, and division | `python calculator.py`       |
| `grade_classifier.py` | Classifies student grades based on scores and displays sorted results                 | `python grade_classifier.py` |
| `guessing_game.py`    | Random number guessing game with attempt tracking                                     | `python guessing_game.py`    |
| `contact_book.py`     | Searches contacts using case-insensitive matching                                     | `python contact_book.py`     |
