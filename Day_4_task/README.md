# Day_4_Task

Day 4 self-practice — Intermediate Python concepts covering Object-Oriented Programming (OOP), Type Hints, CSV & JSON File Handling, Pandas, Inheritance, Dunder Methods, and Project Organization.

## Files

| File                      | What it does                                                                                      | How to run                        |
| ------------------------- | ------------------------------------------------------------------------------------------------- | --------------------------------- |
| `student_report.py`       | Creates a Student class with average calculation, grade generation, and report card display       | `python student_report.py`        |
| `comprehension_drills.py` | Solves multiple list comprehension exercises and demonstrates dict/set comprehensions             | `python comprehension_drills.py`  |
| `file_records.py`         | Creates student records in CSV format, calculates averages, and generates result reports          | `python file_records.py`          |
| `typed_calculator.py`     | Calculator module using type hints, Optional return types, and function docstrings                | `python typed_calculator.py`      |
| `library_system.py`       | Demonstrates inheritance, method overriding, and polymorphism through a library management system | `python library_system.py`        |
| `config_manager.py`       | Reads, writes, and updates configuration data using JSON files                                    | `python config_manager.py`        |
| `pandas_explore.py`       | Uses Pandas DataFrames to analyze student data and answer data-related questions                  | `python pandas_explore.py`        |
| `fraction_class.py`       | Implements a custom Fraction class using dunder methods and operator overloading                  | `python fraction_class.py`        |
| `inventory.py`            | Inventory management system using OOP and CSV persistence                                         | `python inventory.py`             |
| `requirements.txt`        | Contains external Python package dependencies required for the tasks                              | `pip install -r requirements.txt` |
| `students.csv`            | Sample student dataset generated and used by `file_records.py`                                    | Generated automatically           |
| `results.csv`             | Output file containing student averages and grades                                                | Generated automatically           |
| `config.json`             | JSON configuration file generated by `config_manager.py`                                          | Generated automatically           |
| `inventory.csv`           | Inventory data file generated by `inventory.py`                                                   | Generated automatically           |

## Setup

python -m venv .venv
